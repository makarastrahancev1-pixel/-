# 🎮 Битва с Тунг Тунг Сахуром

## 🦸 Персонажи

### 🦈 Тралала Трелоло
- **Способность 1 (ЛКМ):** Обычный удар (3 метра, урон 18)
- **Способность 2 (Пробел):** Супер рывок (полёт 5 секунд с авто-бомбами каждые 1.5с, урон бомбы 20)
- **Способность 3 (E):** Укус челюстью (2 метра, урон 38)
- **Кулдауны:** Способность 2 - 8с, Способность 3 - 12с

### 🟢 Слизь
- **Способность 1:** Отсутствует
- **Способность 2 (Пробел):** Щит (поглощает 1 любую атаку, кулдаун 8с)
- **Способность 3 (E):** Пожиратель (7 метров, урон 50, можно использовать только после поглощения урона щитом, кулдаун 12с)

### 💀 Призыватель
- **Способность 1 (ЛКМ):** Призыв скелетика (кулдаун 1с)
- **Способность 2 (Пробел):** Призыв скелетика (кулдаун 1с)
- **Способность 3 (E):** Призыв скелетика (кулдаун 1с)
- **Скелетики:** Маленькие миньоны с очень малым HP (умирают от 1 удара), атакуют босса (урон 6)

### 👤 Человек
- **Способность 1:** Отсутствует
- **Способность 2 (Пробел):** Супер полёт (4 секунды, скорость x35, кулдаун 10с)
- **Способность 3 (E):** Супер удар (огромный хитбокс 4x4x4, урон 45, **без кулдауна**)

## 🎮 Управление

| Клавиша | Действие |
|---------|----------|
| WASD | Передвижение (W - вперёд, S - назад, A - влево, D - вправо) |
| Мышь | Вращение камеры |
| ЛКМ | Способность 1 |
| Пробел | Способность 2 |
| E | Способность 3 |
| R | Сброс игры |
| Цифры 1, 2, 3 | Альтернативные клавиши способностей |

## 👹 Босс — Тунг Тунг Сахур «Громовой Клык»

- **HP:** 300
- **Обычная атака:** Красные сферы (урон 22, усиленный урон 28 при пентаграмме)
- **Способность «Бита»:** Удар на 3 метра (урон 10-12, может оглушить на 2с, кулдаун 5с)
- **Способность «Пентаграмма»:** Усиление всех характеристик на 5 секунд (скорость x2, урон увеличен, кулдаун 12с)
- **Особенности:** Не умеет летать, не преследует игрока в воздухе

## 🤝 NPC-друг

Автоматически управляемый союзник, который помогает в битве:
- Если вы играете за Тралала Трелоло → друг Слизь
- Если вы играете за Слизь/Призывателя/Человека → друг Тралала Трелоло
- **HP:** 100
- **Способности друга (Тралала Трелоло):** Обычный удар, Укус челюстью, Полёт с бомбами

## 🚀 Установка и запуск

### Локальный запуск
1. Скачайте HTML-файл
2. Откройте его в любом современном браузере (Chrome, Firefox, Edge)
3. Наслаждайтесь игрой!

### Онлайн запуск
Просто откройте HTML-файл на любом хостинге или используйте GitHub Pages.

## 🛠️ Технологии

- **Three.js** (v0.128.0) — 3D графика
- **PointerLockControls** — управление от первого лица
- **CSS2DRenderer** — 2D интерфейс поверх 3D сцены
- Чистый JavaScript (ES6 модули)
- Импорты через CDN (unpkg)

## 📝 Примечания

- Игра использует Pointer Lock API для захвата мыши
- Все хитбоксы атак визуализированы цветными рамками
- Босс не может атаковать игрока в полёте
- Скелетики призывателя умирают от одного удара
- Прогресс не сохраняется между сессиями
